# go_robot_client

## 概述

1. 如果修改机器人数量、进入的游戏，只需修改main.ini的robot标签下的数据
2. 如果是设置具体游戏的参数，修改game.ini中对应游戏的标签中的数据

## 主配置

1. [robot]下的api_addr根据不同的产品，修改对应的二级域名

## 游戏配置

### 通用

frame：帧数，调用update()的频率

### 捕鱼

1. 捕鱼可以设置捕获指定类型的鱼

### 水果机

1. 必须指定room_id，否则无法进入房间

### 阿拉丁

1. 必须指定room_id，否则无法进入房间
